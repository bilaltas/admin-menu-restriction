The best and easiest plugin to restrict admin menu items.

== Description ==
Minimizing the admin page hasn't been easier! The best and easiest plugin to restrict admin menu items for roles. Choose a role, and select what to show. It's that easy.


== Installation ==
1. Login to your website and go to the Plugins section of your admin panel.
2. Click the Add New button.
3. Under Install Plugins, click the Upload link.
4. Select the plugin zip file (admin-menu-restriction.zip) from your computer then click the Install Now button.
5. You should see a message stating that the plugin was installed successfully.
6. Click the Activate Plugin link.


== Screenshots ==
1. Select a role to edit access
2. Choose the menu items that need to be shown
3. Or, just choose the menu items that the selected role shouldn't have access
